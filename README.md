# Portfolio Backend

Node.js backend API for Kritika Singh's portfolio website.

## Features
- Contact form API
- MongoDB integration
- CORS enabled
- Environment configuration

## Setup
```bash
npm install
cp .env.example .env
npm run dev
```

## API Endpoints
- `POST /api/contact` - Submit contact form
- `GET /api/contact` - Get all contacts

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
